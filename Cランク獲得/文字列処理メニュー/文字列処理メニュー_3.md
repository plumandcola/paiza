# 文字列の分割

https://paiza.jp/works/mondai/string_primer/advance_step1

<details><summary>Python3 解答</summary><br>

`str.split(sep)` は、文字列 `str` を `sep` で区切ってlistにする。<br>
参考: https://docs.python.org/ja/3/library/stdtypes.html#str.split

`input().split(",")` とすることで、入力から受け取った文字列を `,` で区切ってlistにできる。<br>
for文を用いてそのlistの中身を出力する。

```python
S = input().split(",")

for s in S:
    print(s)
```

以下のコードは、上のコードを1行に凝縮したもの。<br>
`*` でlistを展開して引数として渡し、`sep = "\n"` により改行区切りで出力している。

```python
print(*input().split(","), sep = "\n")
```

**別解**<br>
`str.replace(old, new)` は、`str` 中の部分文字列 `old` を `new` に置き換えた文字列を返す。<br>
参考: https://docs.python.org/ja/3/library/stdtypes.html#str.replace

```python
print(input().replace(",", "\n"))
```

<br></details>

<details><summary>C++ 解答</summary><br>

1文字ずつ見ていき、`,` が出てきたらそれまでの文字列を出力する。

`S.size()` は `S.length()` でも可。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    int now = 0;
    for (int i = 0; i < S.size(); i++) {
        if (S[i] == ',') {
            cout << S.substr(now, i-now) << endl;
            now = i+1;
        }
    }

    cout << S.substr(now, S.size() - now) << endl;
}
```

**別解**<br>
`for (const auto& c : S)` は `for (char c : S)` でも良いが、これでは毎回 `S` の中身のコピーを作るため時間とメモリが無駄になる。<br>
そのため `&` で参照渡しをしている。<br>
型の部分は `auto` とすることで、こちらが型を指定する必要がなくなり楽になる。<br>
また、`const` とつけることで、間違えて値を変えることがないようにしている。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    for (const auto& c : S) {
        if (c == ',') {
            cout << endl;
        } else {
            cout << c;
        }
    }
    cout << endl;
}
```

**三項演算子ver.**

```python
S = input()

print("YES" if S.isdigit() else "NO")
```

<br></details>

<details><summary>C++ 解答</summary><br>

`S.size()` は `S.length()` でも可。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;

    for (int i = 0; i < S.size(); i++) {
        if (S[i] - '0' < 0 || 9 < S[i] - '0') {
            cout << "NO" << endl;
            return 0;
        }
    }

    cout << "YES" << endl;
}
```

**書き方を少し変えた別解**<br>
`for (const auto& c : S)` は `for (char c : S)` でも良いが、これでは毎回 `S` の中身のコピーを作るため時間とメモリが無駄になる。<br>
そのため `&` で参照渡しをしている。<br>
型の部分は `auto` とすることで、こちらが型を指定する必要がなくなり楽になる。<br>
また、`const` とつけることで、間違えて値を変えることがないようにしている。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S, ans = "YES";
    cin >> S;
    for (const auto& c : S) {
        if (c < '0' || '9' < c) {
            ans = "NO";
        }
    }
    
    cout << ans << endl;
}
```

**別解2**<br>
`all_of` の `std` は `using namespace std` の `std` とは異なるため、`using namespace std` を使えない。(namespaceの衝突が起こる)

```cpp
#include <iostream>
#include <algorithm>

using std::string;

int main() {
    string S;
    std::cin >> S;

    if (all_of(S.cbegin(), S.cend(), isdigit)) {
        std::cout << "YES" << std::endl;
    } else {
        std::cout << "NO" << std::endl;
    }
}
```

**三項演算子ver.**

```cpp
#include <iostream>
#include <algorithm>

using std::string;

int main() {
    string S;
    std::cin >> S;
    std::cout << (all_of(S.cbegin(), S.cend(), isdigit) ? "YES" : "NO") << std::endl;
}
```

# 重複の削除

https://paiza.jp/works/mondai/string_primer/advance_step5

<details><summary>Python3 解答</summary><br>

既に現れた数字を `set` で記録する。<br>
`x` が集合 `s` に含まれるか判定するには `x in s` を、`x` が集合 `s` に含まれていないか判定するには `x not in s` を用いる。<br>
また、要素 `elem` を集合 `set` に追加するには `set.add(elem)` を用いる。<br>
参考: https://docs.python.org/ja/3/library/stdtypes.html#set.add

```python
S = input()

appeared = set()
for s in S:
    if s not in appeared:
        appeared.add(s)
        print(s, end="")
print()
```

```python
S = input()

ans = ""
appeared = set()
for s in S:
    if s not in appeared:
        ans += s
        appeared.add(s)

print(ans)
```

<br></details>

<details><summary>C++ 解答</summary><br>

既に現れた数字を `set` で記録する。<br>
`set.insert` は `pair<iterator,bool>` を返す。2つ目の `bool` は、値が追加されたかどうか。よって、この `bool` がfalseなら既に `set` に含まれていたことを表すため、これを用いる。<br>
`set` 内の順序は問わないため、`unordered_set` でも可。

また、`S.size()` は `S.length()` でも可。

```cpp
#include <iostream>
#include <set>

using namespace std;

int main() {
    string S, ans="";
    cin >> S;
    set<char> appeared;

    for (int i = 0; i < S.size(); i++) {
        auto it = appeared.insert(S[i]);
        if (it.second) {
            ans += S[i];
        }
    }

    cout << ans << endl;
}
```

**別解1-1**<br>
その数字が既に現れたかどうかを `vector` に記録しても良い。<br>
いつも通り、`vector<int> count(10);` は `vector<int> count(10, 0);` でも可、`count.at(c - '0')` は `count[c - '0']` でも可。

`for (const auto& c : S)` は `for (char c : S)` でも良いが、これでは毎回 `S` の中身のコピーを作るため時間とメモリが無駄になる。<br>
そのため `&` で参照渡しをしている。<br>
型の部分は `auto` とすることで、こちらが型を指定する必要がなくなり楽になる。<br>
また、`const` とつけることで、間違えて値を変えることがないようにしている。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    string S;
    vector<int> count(10);
    cin >> S;
    for (const auto& c : S) {
        if (count.at(c - '0') == 0) {
            cout << c;
            count.at(c - '0') += 1;
        }
    }
    cout << endl;
}
```

**別解1-2**<br>
Cの配列を用いる場合。


# パスワード作成

https://paiza.jp/works/mondai/string_primer/advance_step6

<details><summary>Python3 解答</summary><br>

答えを記録する配列を `C` で初期化したいが、厄介なことに `C` が最後に与えられるので、初めに入力を全て受け取ってしまう。<br>
出力の部分の `print(*ans, sep = "")` は、`print("".join(ans))` でも可。

```python
N = int(input())
Q = int(input())
n = [0] * Q
c = [""] * Q
for i in range(Q):
    n[i], c[i] = input().split()
    n[i] = int(n[i]) - 1
C = input()

ans = [C] * N
for i in range(Q):
    ans[n[i]] = c[i]

print(*ans, sep = "")
```

<br></details>

<details><summary>C++ 解答</summary><br>

答えを記録する配列を `C` で初期化したいが、厄介なことに `C` が最後に与えられるので、初めに入力を全て受け取ってしまう。<br>
いつも通り `.at()` は `[]` でも良い。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, Q;
    cin >> N >> Q;
    vector<int> n(Q);
    vector<char> c(Q);
    char C;
    for (int i = 0; i < Q; i++) {
        cin >> n.at(i) >> c.at(i);
    }
    cin >> C;
    
    vector<char> password(N, C);
    for (int i = 0; i < Q; i++) {
        password.at(n.at(i) - 1) = c.at(i);
    }

    for (int i = 0; i < N; i++) {
        cout << password.at(i);
    }
    cout << endl;
}
```

**別解**

```cpp
#include <iostream>
using namespace std;

int main() {
    int N, Q;

    int n[Q];
    char c[Q], C;

    }

    for (int i = 0; i < N; i++) {
        for (int j = 0; j < Q; j++) {
            if (i == n[j] - 1) {
                password += c[j];
                break;
            }

            if (j == Q-1) {
                password += C;
            }
        }
    }

}
```

# 表記の訂正

https://paiza.jp/works/mondai/string_primer/advance_step7

> [!WARNING]
> 問題一覧の画面で問題ごとに表示される"難易度"から分かるように、"文字列処理メニュー"最難関の超難問。しかも誘導が全くない。<br>
> どうしても解けない場合はスキップ推奨。(別にこの問題が解けなくても悪いことはないので。)

<details><summary>ヒント</summary><br>

まずは `S` 中に小数点が含まれるかどうかで場合分けします。

<br></details>

<details><summary>Python3 解答</summary><br>

```python
S = input()

if "." not in S: #小数点がない場合の処理
    i = 0
    while i < len(S):
        if S[i] != "0":
            break
        i += 1

    if i == len(S): #全て 0 の場合
        print(0)
    else:
        print(S[i:])
else:
    ans = ""
    start = 0

    #ミス 1 の処理
    for i in range(len(S)):
        if S[i] != "0":
            if S[i] == ".":
                ans += "0"

            start = i
            break

    #ミス 2 の処理
    last = 0
    for i in range(len(S)):
        if S[i] != "0":
            last = i

    #ミス 3 の処理
    first_dot = S.find(".")
    for i in range(start, last + 1):
        if S[i] != ".":
            ans += S[i]
        elif i == first_dot:
            ans += S[i]

    print(ans)
```

`if "." not in S:` の中は、`print(int(S))` だけでも上手くいきます。<br>
(小数点がない場合は"ミス 1"しかあり得ず、"先頭に必要のない 0 がいくつかついてしまう"のは `int()` で解消できるため。)

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    string S, ans = "";
    int nonzero = -1, start = 0, last;
    bool nopoint = true, until = true;
    cin >> S;
    
    for (int i = S.size() - 1; 0 <= i; i--) {
        if (S[i] == '.') {
            nopoint = false;
        } else if (S[i] != '0') {
            nonzero = i;
        }
    }
    
    if (nopoint) { // 小数点がない場合の処理
        if (nonzero == -1) { // 全て 0 の場合
            cout << S[0] << endl;
        } else {
            cout << S.substr(nonzero) << endl;
        }
        return 0;
    }

    // ミス 1 の処理
    for (int i = 0; i < S.size(); i++) {
        if (S[i] != '0') {
            if (S[i] == '.') { // 1 未満の小数の処理
                ans += '0';
            }
            start = i;
            break;
        }
    }
    
    // ミス 3 の処理
    for (int i = start; i < S.size(); i++) {
        if (S[i] != '.') {
            ans += S[i];
        } else { // S[i] が小数点だった場合
            if (until) {
                ans += S[i];
                until = false;
            }
        }
    }

    // ミス 2 の処理
    for (int i = 0; i < ans.size(); i++) {
        if (ans[i] != '0') {
            last = i;
        }
    }

    cout << ans.substr(0, last + 1) << endl;
}
```

```cpp
        } else if (plus) {
            ans += c - '0';
        } else {
            ans -= c - '0';
        }
    }
    
    cout << ans<< endl;
}
```

# 巨大な数の足し算

https://paiza.jp/works/mondai/string_primer/advance_step11

<details><summary>ヒント</summary><br>

今回の問題では繰り上がりの可能性があるため、下の桁から見て足し算し、答えとなる文字列に入れていきます。<br>
最後に、まだ繰り上がりが残っている場合はそれを文字列に加えることを忘れずに。<br>
また、下の位から順に文字列に入れているため、出力の前に答えとなる文字列を反転させることも忘れずに。

<br></details>

<details><summary>Python3 解答</summary><br>

```python
S = input()
T = input()

ans = ""
N = 0
for i in range(len(S) - 1, -1, -1):
    N += int(S[i]) + int(T[i])
    ans += str(N % 10)
    N //= 10

if N > 0:
    ans += str(N)

print(ans[::-1])
```

**別解**<br>
Pythonでは基本的にオーバーフローが発生しないので、単純に計算させても答えが出ます。

```python
print(int(input()) + int(input()))
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    string S, T, ans = "";
    cin >> S >> T;
    int N = 0;
    for (int i = S.size() - 1; 0 <= i; i--) {
        N += (S[i] - '0') + (T[i] - '0');
        ans += to_string(N % 10);
        N /= 10;
    }
    
    if (N > 0) {
        ans += to_string(N);
    }
    
    reverse(ans.begin(), ans.end());
    cout << ans << endl;
}
```

# 巨大な数のかけ算

<details><summary>ヒント</summary><br>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>

int main() {
```
