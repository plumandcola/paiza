# 部分文字列

https://paiza.jp/works/mondai/string_primer/normal_step1

<details><summary>Python3 解答</summary><br>

「スライス」を用いて部分文字列を出力します。

```python
S = input()
i, j = map(int, input().split())

print(S[i-1:j])
```

<br></details>

<details><summary>C++ 解答</summary><br>

`s.substr(pos, n)` は、`s` の `pos` +1文字目から `n` 文字分の部分文字列を返す。<br>
開始位置のインデックスを0番目からの数字に直すことと、i文字目からj文字目までは `j-i+1` 文字であることに注意。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    int i, j;
    cin >> S >> i >> j;
    cout << S.substr(i-1, j-i+1) << endl;
}
```

# 文字列の挿入

https://paiza.jp/works/mondai/string_primer/normal_step2

<details><summary>Python3 解答</summary><br>

Python3で、文字列中に文字列を挿入することはできません。そのため `S` の `N` 文字目の後ろに `T` を挿入した文字列を、新しい文字列として生成します。<br>
`S` の `N` 文字目まで、`T`、`S` の `N` +1文字目以降を繋げます。<br>
インデックスが0から始めることに注意しつつ、スライスを使いましょう。

```python
S = input()
T = input()
N = int(input())

print(S[:N] + T + S[N:])
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    string S, T;
    int N;
    cin >> S >> T >> N;

    S.insert(N, T);

    cout << S << endl;
}
```

`S.insert(N, T)` を出力しても良い。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S, T;
    int N;
    cin >> S >> T >> N;
    
    cout << S.insert(N, T) << endl;
}
```

# 文字列の書き換え

https://paiza.jp/works/mondai/string_primer/normal_step3

<details><summary>Python3 解答</summary><br>

Python3で、文字列中の文字を変更することはできません。そのため `S` の `i` 文字目を `c` に変更した文字列を、新しい文字列として生成します。<br>
`S` の `i` -1文字目まで、`c`、`S` の `i` +1文字目以降を繋げます。<br>
インデックスが0から始めることに注意しつつ、スライスを使いましょう。

```python
S = input()

i, c = input().split()
i = int(i)

print(S[:i-1] + c + S[i:])
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    int i;
    char c;
    cin >> S >> i >> c;
    S[i-1] = c;
    cout << S << endl;
}
```

**別解1-1**<br>
`replace` を用いる。`c` もstring型でないといけないので注意。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S, c;
    int i;
    cin >> S >> i >> c;
    S.replace(i-1, 1, c);
    cout << S << endl;
}
```

**別解1-2**<br>
`S.replace(i-1, 1, c)` を出力しても良い。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S, c;
    int i;
    cin >> S >> i >> c;
    cout << S.replace(i-1, 1, c) << endl;
}
```

# 大文字から小文字への変換

https://paiza.jp/works/mondai/string_primer/normal_step6

<details><summary>Python3 解答</summary><br>

```python
S = input()
print(S.lower())
```

<br></details>

<details><summary>C++ 解答</summary><br>

`S.size()` は `S.length()` でも可。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    for (int i = 0; i < S.size(); i++) {
        S[i] = tolower(S[i]);
    }
    cout << S << endl;
}
```

**別解**<br>
`for (const auto& c : S)` は `for (char c : S)` でも良いが、これでは毎回 `S` の中身のコピーを作るため時間とメモリが無駄になる。<br>
そのため `&` で参照渡しをしている。<br>
型の部分は `auto` とすることで、こちらが型を指定する必要がなくなり楽になる。<br>
また、`const` とつけることで、間違えて値を変えることがないようにしている。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    for (const auto& c : S) {
        cout << (char)tolower(c);
    }
    cout << endl;
}
```

<details><summary>Python3 解答</summary><br>

1文字ずつ、小文字か大文字かを判定し、小文字なら大文字にして、大文字なら小文字にして出力します。

```python
S = input()

for c in S:
    if c.islower():
        print(c.upper(), end = "")
    else:
        print(c.lower(), end = "")

print()
```

**別解1**

```python
S = input()
print(S.swapcase())
```

**別解2**<br>

https://zenn.dev/today2098/articles/250104-cpp-alphabet_case_change

```python
S = input()

for c in S:
```

<br></details>

<details><summary>C++ 解答</summary><br>

1文字ずつ、小文字か大文字かを判定し、小文字なら大文字にして、大文字なら小文字にして出力します。<br>
`S.size()` は `S.length()` でも可。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    for (int i = 0; i < S.size(); i++) {
        if (isupper(S[i])) {
            S[i] = tolower(S[i]);
        } else {
            S[i] = toupper(S[i]);
        }
    }
    cout << S << endl;
}
```

**別解1**<br>
`for (const auto& c : S)` は `for (char c : S)` でも良いが、これでは毎回 `S` の中身のコピーを作るため時間とメモリが無駄になる。<br>
そのため `&` で参照渡しをしている。<br>
型の部分は `auto` とすることで、こちらが型を指定する必要がなくなり楽になる。<br>
また、`const` とつけることで、間違えて値を変えることがないようにしている。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    for (const auto& c : S) {
        if (islower(c)) {
            cout << (char)toupper(c);
        } else {
            cout << (char)tolower(c);
        }
    }
    cout << endl;
}
```

**別解2**<br>
詳細は以下のサイトを参照<br>
https://zenn.dev/today2098/articles/250104-cpp-alphabet_case_change

```cpp
    }

```

# 文字列の検索

https://paiza.jp/works/mondai/string_primer/normal_step9

<details><summary>Python3 解答</summary><br>

```python
S = input()
T = input()

if T in S:
    print("YES")
else:
    print("NO")
```

**別解**<br>
三項演算子を用いて簡潔に表したもの。

```python
S = input()
T = input()

print("YES" if T in S else "NO")
```

<br></details>

<details><summary>C++ 解答</summary><br>

`.size()` は `.length()` でも可。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S, T;
    cin >> S >> T;
    
    if (S.size() < T.size()) {
        cout << "NO" << endl;
        return 0;
    }
    
    for (int i = 0; i <= S.size() - T.size(); i++) {
        if (S.substr(i, T.size()) == T) {
            cout << "YES" << endl;
            return 0;
        }
    }
    
    cout << "NO" << endl;
}
```

**別解**<br>
`T` が `S` の部分文字列でない場合、`S.find(T)` は `string::npos` を返す。そのため `T` が `S` の部分文字列かどうかは、これらが等しいかどうかで判定できる。<br>
出力の部分を三項演算子で簡潔に表すと、`cout << (S.find(T) == string::npos ? "NO" : "YES") << endl;`

```cpp
#include <iostream>
using namespace std;

int main() {
    string S, T;
    cin >> S >> T;
    
    if (S.find(T) == string::npos) {
        cout << "NO" << endl;
    } else {
        cout << "YES" << endl;
    }
}
```

# 回文判定

https://paiza.jp/works/mondai/string_primer/normal_step11

<details><summary>Python3 解答</summary><br>

出力の部分を三項演算子で簡潔に表すと、`print("YES" if S == S[::-1] else "NO")`

```python
S = input()

if S == S[::-1]:
    print("YES")
else:
    print("NO")
```

**別解**<br>
1文字ずつ判定し、違う部分があれば `NO` と出力してループを抜ける。<br>
`else` の部分は、`break` しなかった(ループを抜けなかった)時に実行される。

```python
S = input()

n = len(S)
for i in range(n//2):
    if S[i] != S[n-i-1]:
        print("NO")
        break
else:
    print("YES")
```

<br></details>

<details><summary>C++ 解答</summary><br>

出力の部分を三項演算子で簡潔に表すと、`cout << (S == T ? "YES" : "NO") << endl;`

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    string S, T;
    cin >> S;
    T = S;
    reverse(T.begin(), T.end());

    if (S == T) {
        cout << "YES" << endl;
    } else {
        cout << "NO" << endl;
    }
}
```

**別解**<br>
`.size()` は `.length()` でも可。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    
    string ans = "YES";
    for (int i = 0; i < S.size() / 2; i++) {
        if (S[i] != S[S.size() - i - 1]) {
            ans = "NO";
            break;
        }
    }
    
    cout << ans << endl;
}
```
