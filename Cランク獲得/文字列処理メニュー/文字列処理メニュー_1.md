# i 文字目の出力

https://paiza.jp/works/mondai/string_primer/basic_step3

<details><summary>ヒント</summary><br>

多くのプログラミング言語ではインデックスが0から始まることに注意。

<br></details>

<details><summary>Python3 解答</summary><br>

```python
S = input()
i = int(input())

print(S[i-1])
```

<br></details>

<details><summary>C++ 解答</summary><br>

`S.at(i-1)` は `S[i-1]` でも可。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    int i;
    cin >> S >> i;
    cout << S.at(i-1) << endl;
}
```

# 文字の検索

https://paiza.jp/works/mondai/string_primer/basic_step6

<details><summary>ヒント</summary><br>

多くのプログラミング言語ではインデックスが0から始まることに注意。

<br></details>

<details><summary>Python3 解答</summary><br>

`str.find(sub)` は、`str` の中で `sub` が現れる最小のインデックスを返します。<br>
参考: https://docs.python.org/ja/3/library/stdtypes.html#str.find

```python
S = input()
c = input()

print(S.find(c) + 1)
```

**別解**<br>
`str.index(sub)` も `str.find(sub)` と同様ですが、`str` に `sub` が含まれない場合はエラーになります。<br>
参考: https://docs.python.org/ja/3/library/stdtypes.html#str.index

```python
S = input()
c = input()

print(S.index(c) + 1)
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    char c;
    cin >> S >> c;
    cout << S.find(c) + 1 << endl;
}
```

# 文字列の連結

https://paiza.jp/works/mondai/string_primer/basic_step7

<details><summary>Python3 解答</summary><br>

文字列の連結は `+` で可能です。

```python
N = int(input())

ans = ""
for _ in range(N):
    ans += input()

print(ans)
```

**別解**<br>
入力から受け取った文字列をその場で出力しても良いです。<br>
しかしPython3の出力は、デフォルトで末尾に改行が入ります(デフォルトは `end='\n'`)。そのため `end = ""` として改行をさせないようにします。<br>
最後に `print()` とすることで、改行しています。

```python
N = int(input())

for _ in range(N):
    print(input(), end = "")

print()
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    string S, ans = "";
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> S;
        ans += S;
    }
    cout << ans << endl;
}
```

**別解**<br>
入力から受け取った文字列をその場で出力しても良いです。<br>
しかし `endl` で改行しないように注意してください。

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    string S;
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> S;
        cout << S;
    }
    cout << endl;
}
```
