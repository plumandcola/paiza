# 文字列の出現率

https://paiza.jp/works/mondai/data_structure/data_structure__dict_step3

<details><summary>Python3 ヒント</summary><br>

条件「S_i は英小文字「a」,「b」, ... ,「z」からなる1文字以上3文字以下の文字列 (1 ≦ i ≦ N)」より、S_iは18278通りで全てに対応できなくはないが、リストで管理するのは難しいし、条件が厳しくなるとさらに難しくなる。

そのためdict型を使おう。

dict型の注意として、`s` を入力から受け取った後で単に `count[s] += 1` としてしまうと、`count` 内に `s` が含まれていない場合にエラーが起きてしまう。`s not in count` で条件分岐を行う。

<br></details>

<details><summary>Python3 解答</summary><br>

```python
N = int(input())
count = {}
for _ in range(N):
    s = input()
    if s not in count:
        count[s] = 1
    else:
        count[s] += 1

for string in sorted(count):
    print(string, count[string])
```

別解<br>
`dict.items()` で、`(key, value)` の一覧を返す。(参考: https://docs.python.org/ja/3/library/stdtypes.html#dict.items)

```python
N = int(input())
count = {}
for _ in range(N):
    s = input()
    if s not in count:
        count[s] = 1
    else:
        count[s] += 1

for key, value in sorted(count.items()):
    print(key, value)
```

別解2<br>
defaultdictを用いると、`count` 内に `s` が含まれているかどうかを判定しなくてもよい。<br>
参考: https://docs.python.org/3/library/collections.html#collections.defaultdict

```python
from collections import defaultdict
N = int(input())
count = defaultdict(int)
for _ in range(N):
    s = input()
    count[s] += 1

for string in sorted(count):
    print(string, count[string])
```

<details><summary>C++ 解答</summary><br>

C++の `map` では要素が昇順に並んでいるため、出力の際に改めてソートする必要はない。

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    int N;
    string S;
    map<string, int> count;
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> S;
        count[S]++;
    }
    
    for (auto it = count.begin(); it != count.end(); it++) {
        cout << it->first << " " << it->second << endl;
    }
}
```

別解<br>
`const auto& p` の部分は `pair<string, int> p` でも良いが、無駄な時間やメモリを消費する。<br>
`&` は"参照渡し"と呼ばれるもの。<br>
`const` とつけることで、間違えて値を変えることがないようにしている。

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    int N;
    string S;
    map<string, int> count;
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> S;
        count[S]++;
    }
    
    for (const auto& p : count) {
        cout << p.first << " " << p.second << endl;
    }
}
```
