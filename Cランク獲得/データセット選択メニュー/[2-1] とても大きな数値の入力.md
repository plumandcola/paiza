# とても大きな数値の入力

https://paiza.jp/works/mondai/data_structure/data_structure__string_step1

<details><summary>C++ 解答</summary><br>

「N をそのまま出力してください。」とのことなので、$`N`$ を整数型にする必要はない。<br>
$`N`$ を文字列として受け取り、それをそのまま出力すれば良い。<br>
条件より $`0 ≦ N ≦ 10^{1,000}`$ なので、文字列としての $`N`$ は1,000文字以内であり問題ない。

```cpp
#include <iostream>
using namespace std;

int main() {
    string N;
    cin >> N;
    cout << N << endl;
}
```

別解1<br>
`getline` を使う。

```cpp
#include <iostream>
using namespace std;

int main() {
    string N;
    getline(cin, N);
    cout << N << endl;
}
```

別解2<br>
Cのchar配列を使う。

```cpp
#include <iostream>
using namespace std;

int main() {
    char N[1100];
    cin >> N;
    cout << N << endl;
}
```

別解3<br>
入力を1文字ずつ処理する。

```cpp
#include <iostream>
using namespace std;

int main() {
    char c;
    while (cin.get(c)) {
        cout << c;
    }
}
```
