<details><summary>Python3 解答</summary><br>

順番や個数を気にせず、存在するかどうかのみを確認したい場合は、set型(集合型)を用いた方が高速。<br>
`x` が `s` に含まれるか判定するには、`x in s` を用いる。<br>
参考: https://docs.python.org/ja/3/library/stdtypes.html#types-set

```python
N, B = map(int, input().split())
A = set(map(int, input().split()))

if B in A:
    print("Yes")
else:
    print("No")
```

別解<br>
set型(集合型)は `{}` を用いて表す。<br>
また、出力の部分は三項演算子を用いて簡潔に表している。

```python
N, B = map(int, input().split())
A = {int(a) for a in input().split()}

print("Yes" if B in A else "No")
```

コードは用意していないが、set型(集合型)ではなくlistにしても良い。

<details><summary>C++ 解答</summary><br>

順番や個数を気にせず、存在するかどうかのみを確認したい場合は、set型(集合型)を用いた方が高速。<br>
`B` が `set` に含まれているかどうかは、`set.find(B) != set.end()` または `set.count(B)` で判定できる。<br>
`s.find(B) != s.end()` の部分は、`s.count(B)` でも可。

```cpp
#include <iostream>
#include <set>
using namespace std;

int main() {
    int N, B, A;
    set<int> s;
    cin >> N >> B;
    for (int i = 0; i < N; i++) {
        cin >> A;
        s.insert(A);
    }
    
    if (s.find(B) != s.end()) {
        cout << "Yes" << endl;
    } else {
        cout << "No" << endl;
    }
}
```

別解1<br>
前の問題"価格の算出","商品の検索"と同様に、`A` が昇順に並んでいる必要はないため、`unordered_set` を使っても良い。<br>
`B` が `unordered_set` に含まれているかどうかの判定に `.count()` を用いている点と、出力に三項演算子を用いている点が、本解と異なる。(`unordered_set` でなく `set` でも同様のコードを用いることができる。)

```cpp
#include <iostream>
#include <unordered_set>
using namespace std;

int main() {
    int N, B, A;
    unordered_set<int> s;
    cin >> N >> B;
    for (int i = 0; i < N; i++) {
        cin >> A;
        s.insert(A);
    }
    
    cout << (s.count(B) ? "Yes" : "No") << endl;
}
```

別解2<br>
`set` を使わない。(問題の趣旨に反しているので非推奨)

```cpp
#include <iostream>
using namespace std;

int main() {
    int N, B, A;
    bool ans = false;
    cin >> N >> B;
    for (int i = 0; i < N; i++) {
        cin >> A;
        if (A == B) {
            ans = true;
            break;
        }
    }
    cout << (ans ? "Yes" : "No") << endl;
}
```
