<details><summary>Python3 解答</summary><br>

要素が存在するかどうかの判定は、set型(集合型)を用いたほうが高速。<br>
しかし、元の数列 `A` は保持しておかないといけないので、入力からの受け取りはlistにする。

要素 `elem` を `set` に追加するには、`set.add(elem)` を用いる。<br>
参考: https://docs.python.org/ja/3/library/stdtypes.html#set.add<br>
`A = list(map(int, input().split()))` の部分は `A = [int(a) for a in input().split()]` でも良い。

"データセット選択メニュー"内では必要ないが、**注意点**<br>
波括弧内にカンマ区切りで要素を列挙することで(`{..., ..., ...}`)、set型(集合型)を生成するが、空のset型(集合型)は `{}` では生成できない。これではdict型が生成される。<br>
空のset型(集合型)の正しい生成方法は、`set()`。

```python
N = int(input())
A = list(map(int, input().split()))

nums = {A[0]}
for i in range(1, N):
    if A[i] in nums:
        print("Yes")
    else:
        nums.add(A[i])
        print("No")
```

<details><summary>C++ 解答</summary><br>

今までの問題を応用して、for文を工夫すれば解けるはず。<br>
解説は省略。

```cpp
#include <iostream>
#include <set>
using namespace std;

int main() {
    int N, A;
    set<int> s;
    cin >> N;
    cin >> A;
    s.insert(A);
    for (int i = 1; i < N; i++) {
        cin >> A;
        if (s.find(A) != s.end()) {
            cout << "Yes" << endl;
        } else {
            cout << "No" << endl;
        }
        s.insert(A);
    }
}
```

**別解**

```cpp
#include <iostream>
#include <unordered_set>
using namespace std;

int main() {
    int N, A;
    unordered_set<int> s;
    cin >> N;
    cin >> A;
    s.insert(A);
    for (int i = 1; i < N; i++) {
        cin >> A;
        cout << (s.count(A) ? "Yes" : "No") << endl;
        s.insert(A);
    }
}
```
