# 数値の出現率

<details><summary>Python3 解答</summary><br>

出力の部分は、list `count` を `*` で展開して引数として渡し、半角スペース区切りで出力に表示しています。<br>
Python3の `print` に複数の引数を渡した際は、デフォルトで半角スペースで区切って出力に表示します(デフォルトが `sep=' '` となっている)。

```python
N = int(input())
A = list(map(int, input().split()))

count = [0] * 10
for a in A:
    count[a] += 1

print(*count)
```

別解1<br>
dict型を用いる。<br>
出力の部分は、`join` を使って半角スペース区切りの文字列にし、出力に表示しています。<br>
`join` の使い方は、`'間に挿入する文字列'.join([連結したい文字列のリスト])` (参考: https://note.nkmk.me/python-string-concat/, https://docs.python.org/ja/3/library/stdtypes.html#str.join)<br>
`join` の引数は文字列でないといけないので、`map(str, ...)` により `...` の要素を文字列にしています。

```python
N = int(input())
A = [int(a) for a in input().split()]

count = {i: 0 for i in range(10)}
for i in range(N):
    count[A[i]] += 1

print(" ".join(map(str, [count[i] for i in range(10)])))
```

別解2<br>
`sequence.count(value)` で、`sequence` 内の `value` を数える。<br>
しかしこのやり方だと、`A` を10回探索することになり、効率が悪い。

```python
N = int(input())
*A, = map(int, input().split())

print(*[A.count(i) for i in range(10)])
```

<details><summary>C++ 解答</summary><br>

`vector<int> count(10);` を `vector<int> count(10, 0);` として、0で初期化することを明示しても良い。<br>
いつも通り、`.at()` は `[]` でも良い。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, A;
    vector<int> count(10);
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> A;
        count.at(A)++;
    }
    
    for (int i = 0; i < 9; i++) {
        cout << count.at(i) << " ";
    }
    cout << count.at(9) << endl;
}
```

別解<br>
出力の処理を、三項演算子を用いて簡潔にしたもの。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, A;
    vector<int> count(10);
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> A;
        count[A]++;
    }
    
    for (int i = 0; i < 10; i++) {
        cout << count[i] << (i < 9 ? " " : "\n");
    }
}
```

別解2<br>
Cの配列を使う。<br>
`int count[10] = {};` としないと0で初期化できず、バグが起こる。<br>
×: `int count[10];` (0で初期化できていない)

```cpp
#include <iostream>
using namespace std;

int main() {
    int N, A;
    int count[10] = {};
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> A;
        count[A]++;
    }
    
    for (int i = 0; i < 9; i++) {
        cout << count[i] << " ";
    }
    cout << count[9] << endl;
}
```

別解3<br>
`array` を使う。<br>
別解で使ったCの配列と同様、`array<int, 10> count = {};` としないと0で初期化できず、バグが起こる。<br>
×: `array<int, 10> count;` (0で初期化できていない)

```cpp
#include <iostream>
#include <array>
using namespace std;

int main() {
    int N, A;
    array<int, 10> count = {};
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> A;
        count[A]++;
    }
    
    for (int i = 0; i < 9; i++) {
        cout << count[i] << " ";
    }
    cout << count[9] << endl;
}
```

別解4<br>
`map` を使う。

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    int N, A;
    map<int, int> count;
    
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> A;
        count[A]++;
    }
    
    for (int i = 0; i <= 9; i++) {
        cout << count[i] << (i < 9 ? " " : "\n");
    }
}
```
