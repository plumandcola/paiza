# 重複の削除

https://paiza.jp/works/mondai/data_structure/data_structure__set_step2

<details><summary>Python3 解答</summary><br>

重複を削除するためにはset型(集合型)を用いると良い。

```python
N = int(input())
A = set(map(int, input().split()))

print(*sorted(A))
```

**別解**<br>
出力の部分は、`join` を使って半角スペース区切りの文字列にし、出力に表示しています。<br>
`join` の使い方は、`'間に挿入する文字列'.join([連結したい文字列のリスト])` (参考: https://note.nkmk.me/python-string-concat/, https://docs.python.org/ja/3/library/stdtypes.html#str.join)<br>
`join` の引数は文字列でないといけないので、`map(str, ...)` により `...` の要素を文字列にしています。

```python
N = int(input())
A = {int(a) for a in input().split()}

print(" ".join(map(str, sorted(A))))
```

<details><summary>C++ 解答</summary><br>

「昇順に出力してください」とあるので、`unordered_set` ではなく `set` が良い。

```cpp
#include <iostream>
#include <set>
using namespace std;

int main() {
    int N, A;
    set<int> s;
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> A;
        s.insert(A);
    }
    
    for (auto it = s.begin(); it != s.end(); it++) {
        cout << (it != s.begin() ? " " : "") << *it;
    }
    cout << endl;
}
```

**別解**<br>
`set` を使わない。<br>
問題文に「数列 A は昇順にソートされています。」とあるため、重複した要素がある場合は、末尾の要素と一致するはずである。<br>
そのため、「Aが空か、末尾の要素と一致するなら追加」を繰り返せば良い。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, A_i;
    vector<int> A;
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> A_i;
        if (A.size() == 0 || A.back() != A_i) {
            A.push_back(A_i);
        }
    }

    for (int i = 0; i < A.size(); i++) {
        cout << A.at(i) << (i < A.size() - 1 ? " " : "\n");
    }
}
```
