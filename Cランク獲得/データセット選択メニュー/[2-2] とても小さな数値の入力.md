# とても小さな数値の入力

https://paiza.jp/works/mondai/data_structure/data_structure__string_step2

<details><summary>C++ 解答</summary><br>

前の"とても大きな数値の入力"と同様、「そのまま出力してください。」とのことなので、$`N`$ を浮動小数点数にする必要はない。<br>
$`N`$ を文字列として受け取り、それをそのまま出力すれば良い。<br>
条件に「$`0 ≦ N < 100`$」「$`N`$ は高々小数第 100 位まで与えられる」とあり、文字列としての $`N`$ は103文字以内であり問題ない。

解答・別解は"[2-1] とても大きな数値の入力"を参照してください。<br>
Cのchar配列を使う別解2に関しては、`char N[1100];` の部分を `char N[110];` としても良いです。
