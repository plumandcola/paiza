# 複数回のランダムアクセス

https://paiza.jp/works/mondai/data_structure/data_structure__array_step2

<details><summary> C++ 解答</summary><br>

`vector` を使い、`vec.at(i)` でアクセス

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, Q, B;
    cin >> N;
    vector<int> A(N);
    for (int i = 0; i < N; i++) {
        cin >> A.at(i);
    }
    cin >> Q;
    for (int i = 0; i < Q; i++) {
        cin >> B;
        cout << A.at(B-1) << endl;
    }
}
```

`vector` を使い、`vec[i]` でアクセス

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, Q, B;
    cin >> N;
    vector<int> A(N);
    for (int i = 0; i < N; i++) {
        cin >> A[i];
    }
    cin >> Q;
    for (int i = 0; i < Q; i++) {
        cin >> B;
        cout << A[B-1] << endl;
    }
}
```
