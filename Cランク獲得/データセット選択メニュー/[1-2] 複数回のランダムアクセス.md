# 複数回のランダムアクセス

https://paiza.jp/works/mondai/data_structure/data_structure__array_step2

<details><summary>Python3 解答</summary><br>

```python
N = int(input())
A = list(map(int, input().split()))

Q = int(input())
B = list(map(int, input().split()))
for b in B:
    print(A[b-1])
```

別解<br>
数列 `B` を受け取る時点で-1することで、添え字を0-indexedにしています。

```python
N = int(input())
A = [int(a) for a in input().split()]

Q = int(input())
B = [int(b) - 1 for b in input().split()]
for i in range(Q):
    print(A[B[i]])
```

<details><summary>C++ 解答</summary><br>

`vector` を使い、`vec.at(i)` でアクセス

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, Q, B;
    cin >> N;
    vector<int> A(N);
    for (int i = 0; i < N; i++) {
        cin >> A.at(i);
    }
    cin >> Q;
    for (int i = 0; i < Q; i++) {
        cin >> B;
        cout << A.at(B-1) << endl;
    }
}
```
