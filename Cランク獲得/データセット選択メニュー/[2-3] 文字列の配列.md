<details><summary>Python3 解答</summary><br>

```python
H, W, r, c = map(int, input().split())
s = [input() for _ in range(H)]

if s[r-1][c-1] == '#':
    print('Yes')
else:
    print('No')
```

別解<br>
出力の部分は三項演算子を用いて簡潔に表している。

```python
H, W, r, c = map(int, input().split())
s = []
for _ in range(H):
    s.append(input())

print("Yes" if s[r-1][c-1] == "#" else "No")
```

<details><summary>C++ 解答</summary><br>

string型の `vector` に `S` を格納する。<br>
`S.at(r-1).at(c-1)` はchar型なので、`"#"` (ダブルクォーテーション)ではなく `'#'` (シングルクォーテーション)とする。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int H, W, r, c;
    cin >> H >> W >> r >> c;
    vector<string> S(H);
    for (int i = 0; i < H; i++) {
        cin >> S.at(i);
    }

    if (S.at(r-1).at(c-1) == '#') {
        cout << "Yes" << endl;
    } else {
        cout << "No" << endl;
    }
}
```

別解<br>
いつも通り、`at()` の部分は `[]` でもよい。<br>
また、if文の代わりに三項演算子を使っても良い。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int H, W, r, c;
    cin >> H >> W >> r >> c;
    vector<string> S(H);
    for (int i = 0; i < H; i++) {
        cin >> S[i];
    }
    
    cout << (S[r-1][c-1] == '#' ? "Yes" : "No") << endl;
}
```

別解3-1<br>
charの2次元配列として受け取る。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int H, W, r, c;
    cin >> H >> W >> r >> c;
    vector<vector<char>> S(H, vector<char>(W));
    for (int i = 0; i < H; i++) {
        for (int j = 0; j < W; j++) {
            cin >> S.at(i).at(j);
        }
    }
    
    if (S.at(r-1).at(c-1) == '#') {
        cout << "Yes" << endl;
    } else {
        cout << "No" << endl;
    }
}
```

別解3-2<br>
これまた `at()` の部分は `[]` でもよい。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int H, W, r, c;
    cin >> H >> W >> r >> c;
    vector<vector<char>> S(H, vector<char>(W));
    for (int i = 0; i < H; i++) {
        for (int j = 0; j < W; j++) {
            cin >> S[i][j];
        }
    }

    cout << (S[r-1][c-1] == '#' ? "Yes" : "No") << endl;
}
```
