<details><summary>Python3 解答</summary><br>

リストの末尾に要素を一つ追加するには `list.append(x)` を、リスト末尾の要素を取り除くには `list.pop()` を用います。<br>
参考: https://docs.python.org/ja/3/tutorial/datastructures.html

`print` の部分は、list `A` を `*` で展開して引数として渡し、半角スペース区切りで出力に表示しています。<br>
Python3の `print` に複数の引数を渡した際は、デフォルトで半角スペースで区切って出力に表示します(デフォルトが `sep=' '` となっている)。

```python
N, Q = map(int, input().split())
A = list(map(int, input().split()))

for _ in range(Q):
    query = list(map(int, input().split()))
    if query[0] == 0:
        #push_back
        x = query[1]
        A.append(x)
    elif query[0] == 1:
        #pop_back
        A.pop()
    elif query[0] == 2:
        #print
        print(*A)
```

**別解**<br>
`print` の部分は、`join` を使って半角スペース区切りの文字列にし、出力に表示しています。<br>
`join` の使い方は、`'間に挿入する文字列'.join([連結したい文字列のリスト])` (参考: https://note.nkmk.me/python-string-concat/, https://docs.python.org/ja/3/library/stdtypes.html#str.join)<br>
`join` の引数は文字列でないといけないので、`map(str, A)` により `A` の要素を文字列にしています。

```python
N, Q = map(int, input().split())
A = [int(a) for a in input().split()]

for _ in range(Q):
    query = [int(q) for q in input().split()]
    if query[0] == 0:
        #push_back
        A.append(query[1])
    elif query[0] == 1:
        #pop_back
        A.pop()
    elif query[0] == 2:
        #print
        print(" ".join(map(str, A)))
```

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, Q, query, x;
    cin >> N >> Q;
    vector<int> A(N);
    for (int i = 0; i < N; i++) {
        cin >> A.at(i);
    }
    
    for (int i = 0; i < Q; i++) {
        cin >> query;
        if (query == 0) {
            cin >> x;
            A.push_back(x);
        } else if (query == 1) {
            A.pop_back();
        } else if (query == 2) {
            for (int j = 0; j < A.size() - 1; j++) {
                cout << A.at(j) << " ";
            }
            cout << A.back() << endl;
        }
    }
}
```

**別解2**<br>
Cの配列を使う。

```cpp
#include <iostream>
using namespace std;

int main() {
    int N, Q, query, x;
    cin >> N >> Q;
    int A[200];
    for (int i = 0; i < N; i++) {
        cin >> A[i];
    }
    
    int size = N;
    
    for (int i = 0; i < Q; i++) {
        cin >> query;
        if (query == 0) {
            cin >> x;
            A[size] = x;
            size++;
        } else if (query == 1) {
            size--;
        } else if (query == 2) {
            for (int j = 0; j < size - 1; j++) {
                cout << A[j] << " ";
            }
            cout << A[size - 1] << endl;
        }
    }
}
```
