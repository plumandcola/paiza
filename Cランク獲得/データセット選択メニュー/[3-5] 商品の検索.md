<details><summary>C++ 解答</summary><br>

`S` が `indices` 内になければ `indices[S] = i` とする、としている。<br>
前の問題"価格の算出"と同様に、`indices.find(T) != indices.end()` の部分を `indices.count(T)` としても良い。<br>
(ちなみにindicesは、インデックスindexの複数形)

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    int N, Q;
    string S, T;
    map<string, int> indices;
    cin >> N >> Q;
    for (int i = 1; i <= N; i++) {
        cin >> S;
        if (indices.find(S) == indices.end()) {
            indices[S] = i;
        }
    }
    
    for (int i = 0; i < Q; i++) {
        cin >> T;
        if (indices.find(T) != indices.end()) {
            cout << indices[T] << endl;
        } else {
            cout << -1 <<endl;
        }
    }
}
```

#include <iostream>

**別解1**<br>
前の問題"価格の算出"と同様に、`S` の探索を1回で済ませるコード。<br>
また、`indices[S] = i` の代わりに `indices.emplace(S, i)` としている。<br>
前の問題"価格の算出"の別解2と同様、出力の部分で三項演算子を用いることで、簡略化もしている。

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    int N, Q;
    string S, T;
    map<string, int> indices;
    cin >> N >> Q;
    for (int i = 1; i <= N; i++) {
        cin >> S;
        if (!indices.count(S)) {
            indices.emplace(S, i);
        }
    }
    
    for (int i = 0; i < Q; i++) {
        cin >> T;
        auto it = indices.find(T);
        cout << (it != indices.end() ? it->second : -1) << endl;
    }
}
```

**別解2**<br>
前の問題"価格の算出"と同様に、`S` が昇順に並んでいる必要はないため、`unordered_map` を使っても良い。<br>
また、`indices.try_emplace(S, i)` を用いることで、"`S` が `indices` に含まれていないなら追加"を簡潔に実装している。

```cpp
#include <iostream>
#include <unordered_map>
using namespace std;

int main() {
    int N, Q;
    string S, T;
    unordered_map<string, int> indices;
    cin >> N >> Q;
    for (int i = 1; i <= N; i++) {
        cin >> S;
        indices.try_emplace(S, i);
    }
    
    for (int i = 0; i < Q; i++) {
        cin >> T;
        auto it = indices.find(T);
        cout << (it != indices.end() ? it->second : -1) << endl;
    }
}
```
