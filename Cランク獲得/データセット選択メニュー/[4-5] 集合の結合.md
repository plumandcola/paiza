<details><summary>Python3 解答</summary><br>

`A | B` は、`A` と `B` の要素からなる新しい集合を返す(`A + B` でも良い)。<br>
参考: https://docs.python.org/ja/3/library/stdtypes.html#types-set

```python
N = int(input())
A = set(map(int, input().split()))
B = set(map(int, input().split()))

C = A | B

print(*sorted(C))
```

**別解1**<br>
`A.union(B)` で、`A` と `B` の和集合を返す。

出力の部分は、`join` を使って半角スペース区切りの文字列にし、出力に表示しています。<br>
`join` の使い方は、`'間に挿入する文字列'.join([連結したい文字列のリスト])` (参考: https://note.nkmk.me/python-string-concat/, https://docs.python.org/ja/3/library/stdtypes.html#str.join)<br>
`join` の引数は文字列でないといけないので、`map(str, ...)` により `...` の要素を文字列にしています。

```python
N = int(input())
A = {int(a) for a in input().split()}
B = {int(b) for b in input().split()}

C = A.union(B)

print(" ".join(map(str, sorted(C))))
```

**別解2**<br>
`A |= B` で、`B` の要素を追加して `A` を更新している(`A += B` は不可)。

```python
N = int(input())
A = set(map(int, input().split()))
B = set(map(int, input().split()))

A |= B

print(*sorted(A))
```

**別解3**<br>
`A.update(B)` で、`B` の要素を追加して `A` を更新している。

```python
N = int(input())
A = {int(a) for a in input().split()}
B = {int(b) for b in input().split()}

A.update(B)

print(" ".join(map(str, sorted(A))))
```

<details><summary>C++ 解答</summary><br>

`s1.merge(s2)` により、`s1` に `s2` の要素を追加している。

```cpp
#include <iostream>
#include <set>
using namespace std;

int main() {
    int N, A, B;
    set<int> s1, s2;
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> A;
        s1.insert(A);
    }
    for (int i = 0; i < N; i++) {
        cin >> B;
        s2.insert(B);
    }
    
    s1.merge(s2);

    for (auto it = s1.begin(); it != s1.end(); it++) {
        cout << (it != s1.begin() ? " " : "") << *it;
    }
    cout << endl;
}
```

**別解**<br>
入力を受け取る段階で全てを1つの集合に入れてしまえば、操作をする必要はない。

```cpp
#include <iostream>
#include <set>
using namespace std;

int main() {
    int N, A;
    set<int> s;
    cin >> N;
    for (int i = 0; i < 2 * N; i++) {
        cin >> A;
        s.insert(A);
    }

    for (auto it = s.begin(); it != s.end(); it++) {
        cout << (it != s.begin() ? " " : "") << *it;
    }
    cout << endl;
}
```
