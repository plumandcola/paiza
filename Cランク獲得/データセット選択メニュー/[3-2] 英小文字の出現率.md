# 英小文字の出現率

https://paiza.jp/works/mondai/data_structure/data_structure__dict_step2

<details><summary>Python3 ヒント</summary><br>

`ord(character)` で、`character` のUnicodeコードポイントが整数で返される。<br>
逆に `chr(codepoint)` で、`codepoint` がUnicodeコードポイントである文字が文字列strで返される。<br>
参考: https://note.nkmk.me/python-chr-ord-unicode-code-point/, https://docs.python.org/ja/3/library/functions.html#ord, https://docs.python.org/ja/3/library/functions.html#chr

<br></details>

<details><summary>Python3 解答</summary><br>

出力の部分は、list `count` を `*` で展開して引数として渡し、半角スペース区切りで出力に表示しています。<br>
Python3の `print` に複数の引数を渡した際は、デフォルトで半角スペースで区切って出力に表示します(デフォルトが `sep=' '` となっている)。

```python
N = int(input())
S = input()

count = [0] * 26
for c in S:
    count[ord(c) - 97] += 1

print(*count)
```

**別解1**<br>
dict型を用いる。<br>
出力の部分は、`join` を使って半角スペース区切りの文字列にし、出力に表示しています。<br>
`join` の使い方は、`'間に挿入する文字列'.join([連結したい文字列のリスト])` (参考: https://note.nkmk.me/python-string-concat/, https://docs.python.org/ja/3/library/stdtypes.html#str.join)<br>
`join` の引数は文字列でないといけないので、`map(str, ...)` により `...` の要素を文字列にしています。

```python
N = int(input())
S = input()

count = {chr(x): 0 for x in range(ord("a"), ord("z")+1)}
for i in range(N):
    count[S[i]] += 1

print(" ".join(map(str, [count[chr(x)] for x in range(ord("a"), ord("z")+1)])))
```

**別解2**<br>
`sequence.count(value)` で、`sequence` 内の `value` を数える。<br>
しかしこのやり方だと、`S` を26回探索することになり、効率が悪い。

```python
N = int(input())
S = input()

print(*[S.count(chr(x)) for x in range(ord("a"), ord("z")+1)])
```

<details><summary>C++ 解答</summary><br>

`count` の0に `'a'` の個数、1に `'b'` の個数、…、25に `'z'` の個数を格納する。<br>
char型で受け取ることで、`S` から1文字ずつ受け取ることができる。`c - 'a'` は、`c` が `'a'` のときは0、`'b'` のときは1、…、`'z'` のときは25、となる。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N;
    char c;
    vector<int> count(26);
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> c;
        count.at(c - 'a')++;
    }
    
    for (int i = 0; i < 25; i++) {
        cout << count.at(i) << " ";
    }
    cout << count.at(25) << endl;
}
```

**別解**<br>
`map` を使う。<br>
出力の処理を、三項演算子を用いて簡潔にした。

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    int N;
    string S;
    map<char, int> count;
    cin >> N >> S;
    for (char c : S) {
        count[c]++;
    }
    
    for (char c = 'a'; c <= 'z'; c++) {
        cout << count[c] << (c != 'z' ? " " : "\n");
    }
}
```

Cの配列や `array` を使った別解は省略。
