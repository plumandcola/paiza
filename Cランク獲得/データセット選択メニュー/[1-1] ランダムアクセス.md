# ランダムアクセス

https://paiza.jp/works/mondai/data_structure/data_structure__array_step1

<details><summary> C++ 解答</summary><br>

`vector` を使い、`vec.at(i)` でアクセス

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    vector<int> A(N);
    for (int i = 0; i < N; i++) {
        cin >> A.at(i);
    }
    cout << A.at(M-1) << endl;
}
```

`vector` を使い、`vec[i]` でアクセス

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    vector<int> A(N);
    for (int i = 0; i < N; i++) {
        cin >> A[i];
    }
    cout << A[M-1] << endl;
}
```

Cの配列を使用

```cpp
#include <iostream>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    int A[N];
    for (int i = 0; i < N; i++) {
        cin >> A[i];
    }
    cout << A[M-1] << endl;
}
```
