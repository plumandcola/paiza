# 最大値と最小値

https://paiza.jp/works/mondai/data_structure/data_structure__array_step3

<details><summary>Python3 解答</summary><br>

```python
A, B, C = map(int, input().split())
print(max(A, B, C) - min(A, B, C))
```

**別解1**<br>
`A`, `B`, `C` をリストに格納

```python
nums = list(map(int, input().split()))
print(max(nums) - min(nums))
```

**別解2-1**<br>
`A`, `B`, `C` をリストに格納した後、sort

```python
nums = list(map(int, input().split()))
nums.sort()
print(nums[-1] - nums[0])
```

**別解2-2**<br>
`A`, `B`, `C` をリストに格納する時にsort

```python
nums = sorted(map(int, input().split()))
print(nums[len(nums) - 1] - nums[0])
```

<details><summary>C++ 解答</summary><br>

`max`, `min` を使う。

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int A, B, C;
    cin >> A >> B >> C;

    cout << max({A, B, C}) - min({A, B, C}) << endl;
}
```

**別解1-1**<br>
`vector` に対して `max_element`, `min_element` を使う。<br>
`vec.at(i)` は `vec[i]` でも可。

```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main() {
    vector<int> vec(3);
    for (int i = 0; i < 3; i++) {
        cin >> vec.at(i);
    }

    cout << *max_element(vec.begin(), vec.end())
     - *min_element(vec.begin(), vec.end()) << endl;
}
```

**別解1-2**<br>
Cの配列に対して `max_element`, `min_element` を使う。

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int arr[3];
    for (int i = 0; i < 3; i++) {
        cin >> arr[i];
    }

    cout << *max_element(arr, arr + 3)
     - *min_element(arr, arr + 3) << endl;
}
```

**別解1-3**<br>
`array` に対して `max_element`, `min_element` を使う。

```cpp
#include <iostream>

```

**別解2-1**<br>
`vector` をsort。

```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main() {
    vector<int> vec(3);
    for (int i = 0; i < 3; i++) {
        cin >> vec.at(i);
    }

    sort(vec.begin(), vec.end());

    cout << vec.at(2) - vec.at(0) << endl;
}
```

**別解2-2**<br>
`vec.at(i)` は `vec[i]` でも可。`vec.begin()` は `begin(vec)` でも、`vec.end()` は `end(vec)` でも可。

for文の中身が1行なら `{}` は不要。しかし読みにくいので避けるべき。

```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main() {
    vector<int> vec(3);
    for (int i = 0; i < 3; i++) cin >> vec[i];

    sort(begin(vec), end(vec));

    cout << vec[2] - vec[0] << endl;
}
```

**別解3**<br>
if文を使った条件分岐でゴリ押し。<br>
以下のコードでは、`A`, `B`, `C` のうちの最大値を `M` に、最小値を `m` に格納しています。

```cpp
#include <iostream>
using namespace std;

int main() {
    int A, B, C;
    cin >> A >> B >> C;

    int M = A, m = A;

    if (B > M) M = B;
    if (C > M) M = C;
    if (B < m) m = B;
    if (C < m) m = C;

    cout << M - m << endl;
}
```
