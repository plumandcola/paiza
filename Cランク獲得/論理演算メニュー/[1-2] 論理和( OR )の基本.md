# 論理和( OR )の基本

https://paiza.jp/works/mondai/logical_operation/logical_operation__basic_step2

<details><summary>Python3 解答</summary><br>

ブール演算子(論理演算子) `or` もしくはビット演算子 `|` を用います。

```python
A, B = map(int, input().split())
print(A or B)
```

```python
A, B = map(int, input().split())
print(A | B)
```

<details><summary>C++ 解答</summary><br>

前の問題"論理積( AND )の基本"と同様、`()` をつけないと、`<<` がシフト演算子として認識されてしまい、エラーになる。<br>
以下の解答ではビット演算子 `|` を用いているが、論理演算子 `||` や `or` を用いても良い。

```cpp
#include <iostream>
using namespace std;

int main() {
    int A, B;
    cin >> A >> B;
    cout << (A | B) << endl;
}
```
