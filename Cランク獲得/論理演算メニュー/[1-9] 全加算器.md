# 全加算器

https://paiza.jp/works/mondai/logical_operation/logical_operation__basic_step9

<details><summary>Python3 解答</summary><br>

`A` + `B` + `C1` = (`A` + `B`) + `C1` と考える。<br>
`C2` は、`A` + `B` の時点で繰り上がるか、`A` + `B` の1桁目が1かつ、`C1` が1の時に1となる。そのため `C2 = (A and B) or ((A ^ B) and C1)` と表せる。<br>
次に `S` は、`(A ^ B) ^ C1` = `A ^ B ^ C1` と表せる。

また、`and` の部分は `&` を、`or` の部分は `|` を用いても良い。

```python
A, B, C1 = map(int, input().split())
C2 = (A and B) or ((A ^ B) and C1)
S = A ^ B ^ C1
print(C2, S)
```

```python
A, B, C1 = map(int, input().split())
C2 = (A & B) | ((A ^ B) & C1)
S = A ^ B ^ C1
print(C2, S)
```

<details><summary>C++ 解答</summary><br>

`A` + `B` + `C1` = (`A` + `B`) + `C1` と考える。<br>
`C2` は、`A` + `B` の時点で繰り上がるか、`A` + `B` の1桁目が1かつ、`C1` が1の時に1となる。そのため `C2 = (A & B) | ((A ^ B) & C1)` と表せる。<br>
次に `S` は、`(A ^ B) ^ C1` = `A ^ B ^ C1` と表せる。<br>
半角スペースの入れ忘れに注意。

```cpp
#include <iostream>
using namespace std;

int main() {
    int A, B, C1;
    cin >> A >> B >> C1;
    int C2 = (A & B) | ((A ^ B) & C1);
    int S = A ^ B ^ C1;
    cout << C2 << " " << S << endl;
}
```
