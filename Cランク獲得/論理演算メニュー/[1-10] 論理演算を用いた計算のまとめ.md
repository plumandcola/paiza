# 論理演算を用いた計算のまとめ

https://paiza.jp/works/mondai/logical_operation/logical_operation__basic_boss

<details><summary>Python3 解答</summary><br>

提示された式をそのまま実装すれば正解できる。

```python
A, B, C, D = map(int, input().split())
print(int(not((not A and not B) or not C) ^ D))
```

paiza公式の解説の通り、ド・モルガンの法則を用いて式変形することもできる。(式変形はここでは省略。Wikipediaのド・モルガンの法則についてのページや、paiza公式の解説を参照。)

```python
A, B, C, D = map(int, input().split())
print(((A or B) and C) ^ D)
```

```python
A, B, C, D = map(int, input().split())
print(((A | B) & C) ^ D)
```

<details><summary>C++ 解答</summary><br>

提示された式をそのまま実装すれば正解できる。

```cpp
#include <iostream>
using namespace std;

int main() {
    int A, B, C, D;
    cin >> A >> B >> C >> D;
    cout << (!((!A & !B) | !C) ^ D) << endl;
}
```

paiza公式の解説の通り、ド・モルガンの法則を用いて式変形することもできる。(式変形はここでは省略。Wikipediaのド・モルガンの法則についてのページや、paiza公式の解説を参照。)

```cpp
#include <iostream>
using namespace std;

int main() {
    int A, B, C, D;
    cin >> A >> B >> C >> D;
    cout << (((A | B) & C) ^ D) << endl;
}
```
