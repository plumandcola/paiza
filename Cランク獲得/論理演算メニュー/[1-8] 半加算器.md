# 半加算器

https://paiza.jp/works/mondai/logical_operation/logical_operation__basic_step8

<details><summary>Python3 解答</summary><br>

`C` は `A` も `B` も1のときのみ1、それ以外の時は0。つまり `C = A and B` が成り立つ。<br>
`S` は `A` と `B` のどちらか一方のみ1のとき1、それ以外の時は0。つまり `S = A ^ B` が成り立つ。

また、"論理積( AND )の基本"で述べた通り、`and` の部分は `&` を用いても良い。

```python
A, B = map(int, input().split())
C = A and B
S = A ^ B
print(C, S)
```

<details><summary>C++ 解答</summary><br>

`C` は `A` も `B` も1のときのみ1、それ以外の時は0。つまり `C = A & B` が成り立つ。<br>
`S` は `A` と `B` のどちらか一方のみ1のとき1、それ以外の時は0。つまり `S = A ^ B` が成り立つ。<br>
半角スペースの入れ忘れに注意。

```cpp
#include <iostream>
using namespace std;

int main() {
    int A, B;
    cin >> A >> B;
    int C = A & B;
    int S = A ^ B;
    cout << C << " " << S << endl;
}
```
