# 要素数の出力

https://paiza.jp/works/mondai/array_primer/array_primer__1dmatrix_output_step1

<details><summary>Python3 解答</summary><br>

```python
a = [5, 1, 3, 4, 5, 12, 6, 8, 1, 3]

print(len(a))
```

<br></details>

<details><summary>C++ 解答</summary><br>

`a.size()` は `size(a)` でも可。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> a = {5, 1, 3, 4, 5, 12, 6, 8, 1, 3};
    cout << a.size() << endl;
}
```

# 全要素の出力

https://paiza.jp/works/mondai/array_primer/array_primer__1dmatrix_output_step2

<details><summary>ヒント</summary><br>

for文を用いて、配列内の要素を出力しましょう。<br>
多くのプログラミング言語ではインデックスが0から始まることに注意。

<br></details>

<details><summary>Python3 解答</summary><br>

```python
a = [5, 1, 3, 4, 5, 12, 6, 8, 1, 3]

for a_i in a:
    print(a_i)
```

`len(a)` は `10` としても良いが、`len(a)` としておけば、listの要素数が変わっても使い回せる点が良い。

```python
a = [5, 1, 3, 4, 5, 12, 6, 8, 1, 3]

for i in range(len(a)):
    print(a[i])
```

`*` でlistを展開して引数として渡し、`sep = "\n"` により改行区切りで出力しても良い。

```python
a = [5, 1, 3, 4, 5, 12, 6, 8, 1, 3]

print(*a, sep = "\n")
```

```python
print(*[5, 1, 3, 4, 5, 12, 6, 8, 1, 3], sep = "\n")
```

listにせずに `print` の引数に渡しても良い。(問題の趣旨に反しているので非推奨)

```python
print(5, 1, 3, 4, 5, 12, 6, 8, 1, 3, sep = "\n")
```

`join` を使って改行区切りの文字列にし、出力に表示しても良い。<br>
`join` の使い方は、`'間に挿入する文字列'.join([連結したい文字列のリスト])` (参考: https://note.nkmk.me/python-string-concat/, https://docs.python.org/ja/3/library/stdtypes.html#str.join)<br>
`join` の引数は文字列でないといけないので、`map(str, a)` により `a` の要素を文字列にしている。

```python
a = [5, 1, 3, 4, 5, 12, 6, 8, 1, 3]

print("\n".join(map(str, a)))
```

<br></details>

<details><summary>C++ 解答</summary><br>

`a.at(i)` は `a[i]`、`a.size()` は `size(a)` でも可。

for文の終了条件は `i < 10` としても良いが、`i < a.size()` としておけば、配列の要素数が変わっても使い回せる点が良い。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> a = {5, 1, 3, 4, 5, 12, 6, 8, 1, 3};
    for (int i = 0; i < a.size(); i++) {
        cout << a.at(i) << endl;
    }
}
```

`for (const auto& a_i : a)` は `for (int a_i : a)` でも良いが、これでは毎回 `a` の中身のコピーを作るため時間とメモリが無駄になる。<br>
そのため `&` で参照渡しをしている。<br>
型の部分は `auto` とすることで、こちらが型を指定する必要がなくなり楽になる。<br>
また、`const` とつけることで、間違えて値を変えることがないようにしている。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> a = {5, 1, 3, 4, 5, 12, 6, 8, 1, 3};
    for (const auto& a_i : a) {
        cout << a_i << endl;
    }
}
```

**別解1-1**<br>
Cの配列

```cpp
#include <iostream>
using namespace std;

int main() {
    int a[] = {5, 1, 3, 4, 5, 12, 6, 8, 1, 3};
    for (int i = 0; i < size(a); i++) {
        cout << a[i] << endl;
    }
}
```

**別解1-2**<br>
Cの配列

```cpp
#include <iostream>
using namespace std;

int main() {
    int a[] = {5, 1, 3, 4, 5, 12, 6, 8, 1, 3};
    for (const auto& a_i : a) {
        cout << a_i << endl;
    }
}
```

**別解2-1**<br>
`array`<br>
`a.at(i)` は `a[i]`、`a.size()` は `size(a)` でも可。

```cpp
#include <iostream>
#include <array>
using namespace std;

int main() {
    array<int, 10> a = {5, 1, 3, 4, 5, 12, 6, 8, 1, 3};
    for (int i = 0; i < a.size(); i++) {
        cout << a.at(i) << endl;
    }
}
```

**別解2-2**

```cpp

#include <array>
using namespace std;

int main() {
    array<int, 10> a = {5, 1, 3, 4, 5, 12, 6, 8, 1, 3};
    for (const auto& a_i : a) {
        cout << a_i << endl;
    }

```

**別解2-3**

```cpp

    array<int, 10> a = {5, 1, 3, 4, 5, 12, 6, 8, 1, 3};
    for_each(a.begin(), a.end(), [](int x) {
        cout << x << endl;
    });
}
```
