# 標準入力 - 文字列を半角スペースで分割4

https://paiza.jp/works/mondai/cgc_problems_012/cgc_problems__pr14_lv12

<details><summary>C++ 解答</summary><br>

<!-- 問題
#include <iostream>
#include <string>
#include <sstream>
using namespace std;
// ----------------------------------------
// [標準入力 - 文字列を半角スペースで分割4]
// ----------------------------------------
// 文字列をスペースで分割して
// その5つ目を出力してください

int main(void) {
 	string t1, t2, t3, t4, t5, str = "PHP Python Ruby Java C#";
	stringstream ss;
	ss << str;
    ss >> t1 >> t2 >> t3 >> t4 >> t5;
    cout <<  << endl;
}
-->

```cpp
#include <iostream>
#include <string>
#include <sstream>
using namespace std;
// ----------------------------------------
// [標準入力 - 文字列を半角スペースで分割4]
// ----------------------------------------
// 文字列をスペースで分割して
// その5つ目を出力してください

int main(void) {
 	string t1, t2, t3, t4, t5, str = "PHP Python Ruby Java C#";
	stringstream ss;
	ss << str;
    ss >> t1 >> t2 >> t3 >> t4 >> t5;
    cout << t5 << endl;
}
```

**別解**

```cpp
#include <iostream>
#include <string>
#include <sstream>
using namespace std;

int main() {
 	string t, str = "PHP Python Ruby Java C#";
	stringstream ss;
	ss << str;
	for (int i = 0; i < 5; i++) {
	    ss >> t;
	}
    cout << t << endl;
}
```
