```cpp
#include <iostream>
#include <string>
using namespace std;
// ----------------------------------------
// [文字列 - 長さ1]
// ----------------------------------------
// 文字列sの長さを出力してください

int main(void) {
	string s = "Hello paiza";
	
	cout << s.length() << endl;
}
```

`s.length()` は `s.size()` でも可。

<details><summary>文字列 - 部分文字列の取得 1 C++ 解答</summary><br>

https://paiza.jp/works/mondai/cgc_problems_005/cgc_problems__pr15_lv05

```cpp
#include <iostream>
#include <string>
using namespace std;
// ----------------------------------------
// [文字列 - 部分文字列の取得1]
// ----------------------------------------
// 文字列sの左から10個目から3文字分の
// 部分文字列を出力してください

int main(void) {
	string s = "abcdefghijklmnopqrstuvwxyz";
	
	// n個目の文字からm文字分の部分文字列はs.substr(n-1, m)で取得できます
	cout << s.substr(9, 3) << endl;
}
```

