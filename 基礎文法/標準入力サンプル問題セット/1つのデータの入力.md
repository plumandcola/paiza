<details><summary>Python3</summary>

`input()` で標準入力から1行読み取って変数 `S` に代入し、それを `print(S)` で標準出力に表示する。

```python
S = input()
print(S)
```

今回は標準入力から読み取った文字列を後で使わないので、変数に代入せずに直接 `print()` で標準出力に表示してもよい。

```python
print(input())
```

</details>

<details><summary>C++</summary>

`string S;` で文字列型の変数 `S` を宣言<br>
→ `cin >> S;` で標準入力から(空白区切りで)1つのデータ読み取って `S` に代入<br>
→ `cout << S << endl;` で標準出力に `S` と改行を表示

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    cout << S << endl;
}
```

</details>
