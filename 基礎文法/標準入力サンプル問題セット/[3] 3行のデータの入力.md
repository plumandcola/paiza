# 3行のデータの入力

https://paiza.jp/works/mondai/stdin/stdin_3_line

<details><summary>Python3</summary><br>

前の問題"1行のデータの入力"で行ったことを、変数の名前を変えて3回繰り返しましょう。

```python
A = input()
print(A)

B = input()
print(B)

C = input()
print(C)
```

入力を全て読み取ってから出力を表示しても構いません。

```python
A = input()
B = input()
C = input()

print(A)
print(B)
print(C)
```

<br></details>

<details><summary>C++</summary><br>

前の問題"1行のデータの入力"で行ったことを、変数の名前を変えて3回繰り返しましょう。

```cpp
#include <iostream>
using namespace std;

int main() {
    string A, B, C;

    getline(cin, A);
    cout << A << endl;

    getline(cin, B);
    cout << B << endl;
    
    getline(cin, C);
    cout << C << endl;
}
```

入力を全て読み取ってから出力を表示しても構いません。

```cpp
#include <iostream>
using namespace std;

int main() {
    string A, B, C;

    getline(cin, A);
    getline(cin, B);
    getline(cin, C);

    cout << A << endl;
    cout << B << endl;
    cout << C << endl;
}
```

<br></details>

<details><summary>応用: 楽をする方法</summary><br>

for文もしくはwhile文を使いましょう。

```python
for i in range(3):
    S = input()
    print(S)
```

```python
i = 0
while i < 3:
    S = input()
    print(S)
    i += 1
```

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;

    for (int i = 0; i < 3; i++) {
        getline(cin, S);
        cout << S << endl;
    }
}
```

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;

    int i = 0;
    while (i < 3) {
        getline(cin, S);
        cout << S << endl;
        i++;
    }
}
```
