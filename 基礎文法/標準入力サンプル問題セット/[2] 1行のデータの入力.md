<details><summary>1つ前の問題"1つのデータの入力"との違い</summary><br>

"条件"に太字で書かれている通り、入力に半角スペースが含まれます。

<br></details>

<details><summary>Python3</summary><br>

Pythonでは `input()` を使うと、標準入力から半角スペースも含めて1行を読み取れる。
"1つのデータの入力"と同じコードで正解になる。

```python
S = input()
print(S)
```

別解

```python
print(input())
```

<br></details>

<details><summary>C++</summary><br>

Python3の `input()` とは異なり、C++の `cin >> S;` では**空白区切りで**1つのデータを読み取る。<br>
つまり、今回のように入力に半角スペースが含まれていると、本来は1行全てを読み取ってほしいのに、半角スペースの直前までしか読み取ってくれない。<br>
1行全てを読み取るには、`getline(cin, S);` を使う。
ほかは"1つのデータの入力"と同じ。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    getline(cin, S);
    cout << S << endl;
}
```
