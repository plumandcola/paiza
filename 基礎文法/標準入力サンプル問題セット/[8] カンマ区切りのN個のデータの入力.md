# カンマ区切りのN個のデータの入力

https://paiza.jp/works/mondai/stdin/stdin_comma_n

"N行のデータの入力"と"カンマ区切りの3つのデータの入力"を組み合わせて考えれば正解です。

<details><summary>Python3 解答</summary><br>

```python
N = int(input())
S = input().split(",")

for x in S:
    print(x)
```

**後半の別解**(解説は"3つのデータの入力"参照)

```python
for i in range(N):
    print(S[i])
```

```python
print(*S, sep = "\n")
```

```python
print("\n".join(S))
```

**全体の別解**(解説は"3つのデータの入力"参照)

```python
N = int(input())
S = input()

print(S.replace(",", "\n"))
```

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;

    string dummy;
    getline(cin, dummy);

    string S;
    for (int i = 0; i < N - 1; ++i) {
        getline(cin, S, ',');
        cout << S << endl;
    }
    getline(cin, S);
    cout << S << endl;
}
```

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;

    cin.ignore();

    string S;
    int i = 0;
    while (i < N - 1) {
        getline(cin, S, ',');
        cout << S << endl;
        i++;
    }
    getline(cin, S);
    cout << S << endl;
}
```
