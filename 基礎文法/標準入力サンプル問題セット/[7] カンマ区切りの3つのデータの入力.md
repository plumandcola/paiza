# カンマ区切りの3つのデータの入力

https://paiza.jp/works/mondai/stdin/stdin_comma_3

<details><summary>Python3 解答</summary><br>

`split()` の `()` の中に文字を指定すると、その文字で区切ってlist型にできます。(文字を指定しなければ、空白文字で区切ります。)<br>
そのため `input().split(",")` とすれば、入力から受け取った文字列をカンマで区切ってlist型にできます。<br>
あとは今までの問題と同じです。

```python
S = input().split(",")

for x in S:
    print(x)
```

**後半の別解**(解説は"3つのデータの入力"参照)

```python
for i in range(3):
    print(S[i])
```

```python
print(*S, sep = "\n")
```

```python
print("\n".join(S))
```

**全体の別解**(解説は"3つのデータの入力"参照)

```python
S = input()
print(S.replace(",", "\n"))
```

<details><summary>C++ 解答</summary><br>

`getline(cin, S, ',');` とすれば、カンマまでを受け取ってくれます。<br>
3個目はカンマがないので、`getline(cin, S);` としましょう。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    for (int i = 0; i < 2; ++i) {
        getline(cin, S, ',');
        cout << S << endl;
    }
    getline(cin, S);
    cout << S << endl;
}
```

while文を使ってもいいです。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    int i = 0;
    while (i < 2) {
        getline(cin, S, ',');
        cout << S << endl;
        i++;
    }
    getline(cin, S);
    cout << S << endl;
}
```
