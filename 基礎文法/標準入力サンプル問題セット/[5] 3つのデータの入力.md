<details><summary>Python3 解答</summary><br>

Pythonの `input()` は1行単位で入力を読み取るため、1行に複数のデータがある場合は `split()` を使って空白で分割します。<br>
文字列(例えば変数名を `str` とする)に対して `str.split()` とすると、文字列( `str` )を空白で区切ってlist型にできます。<br>
そのため `input().split()` とすれば、入力から受け取った文字列を空白で区切ってlist型にできます。<br>
for文を使ってlistの中身を一つずつ取り出し、 `print()` で出力に表示すれば正解です。

```python
S = input().split()

for x in S:
    print(x)
```

後半は以下のようにindexを使って書いても正解です。

```python
for i in range(3):
    print(S[i])
```

以下の書き方では、list `S` を `*` で展開して引数として渡し、`sep = "\n"` (`"\n"` は改行)と書くことにより、改行区切りで出力に表示しています。<br>
(`print(S[0], S[1], ..., S[len(S)-1], sep = "\n")`)と同じ

```python
print(*S, sep = "\n")
```

以下の書き方では、list `S` の要素を `"\n"` で結合した文字列を `join` により作り、それを出力に表示しています。<br>
`join` の使い方は、`'間に挿入する文字列'.join([連結したい文字列のリスト])` (参考: https://note.nkmk.me/python-string-concat/, https://docs.python.org/ja/3/library/stdtypes.html#str.join)

```python
print("\n".join(S))
```

全体を以下のように書いても正解です。<br>
今までの `S = input().split()` ではなく `S = input()` と書いているため、ここでの `S` はlistでなく文字列型であることに注意してください。<br>
この問題では出力すべき文字列が空白区切りで与えられるため、`" "` を `"\n"` に置き換えれば正解になります。<br>
`str.replace(old, new)` は、`str` 中の部分文字列 `old` を `new` に置き換えた文字列を返します。ここでは `" "` を `"\n"` に置き換えています。<br>
参考: https://docs.python.org/ja/3/library/stdtypes.html#str.replace

```python
S = input()

print(S.replace(" ", "\n"))
```

<details><summary>C++ 解答</summary><br>

`cin >> S` とすれば、勝手に空白で区切って受け取ってくれます。

```cpp
#include <iostream>
using namespace std;

int main() {
    string A, B, C;
    cin >> A >> B >> C;
    cout << A << endl;
    cout << B << endl;
    cout << C << endl;
}
```

for文やwhile文を使ってもいいです。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    for (int i = 0; i < 3; ++i) {
        cin >> S;
        cout << S << endl;
    }
}
```

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    int i = 0;
    while (i < 3) {
        cin >> S;
        cout << S << endl;
        i++;
    }
}
```
