<details><summary>ヒント2</summary><br>

同じ処理を繰り返し行うには、**for文**や**while文**が使えました。

<br></details>

<details><summary>Python3 ヒント3</summary><br>

以下のコードではプログラムがうまく動かず、エラーが出てしまいます。

```python
N = input()

for i in range(N):
    S = input()
    print(S)
```

標準エラー出力
```
TypeError: 'str' object cannot be interpreted as an integer
```

str(文字列型)を整数としては扱えないと言っています。<br>
`N` がstr(文字列型)になっているようです。これを整数型に直すにはどうすればよいでしょうか？

<br></details>

<details><summary>Python3 解答</summary><br>

`int()` の `()` の中に、整数型に直したいものを入れ、変数に代入しましょう。<br>
変数の名前を同じにしても、上書きされるだけでエラーにはなりません。ただし、変数の中身は上書きされてしまって、元のデータはなくなるので気をつけましょう。

```python
N = input()
N = int(N)

for i in range(N):
    S = input()
    print(S)
```

入力を読み取る段階で `int()` を使ってもいいです。なんならこちらの方がコンパクトなので、ふつうは1行にまとめます。

```python
N = int(input())

for i in range(N):
    S = input()
    print(S)
```

while文を使った場合

```python
N = int(input())

i = 0
while i < N:
    S = input()
    print(S)
    i += 1
```

<details><summary>C++ 解答</summary><br>

変数 `N` をint型として宣言し、標準入力から読み取った数字を `N` に代入しましょう。<br>
cinで入力を読み取ると、`N` の後ろにある空白文字(今回は改行)が残ってしまいます。<br>
そのため、`cin.ignore();` を使うか、paiza公式の解答例のように変数 `dummy` を宣言し、`getline` を使って改行文字を取り除きましょう。

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;

    string dummy;
    getline(cin, dummy);

    string S;
    for (int i = 0; i < N; i++) {
        getline(cin, S);
        cout << S << endl;
    }
}
```

while文を使った場合

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;

    cin.ignore();

    string S;
    int i = 0;
    while (i < N) {
        getline(cin, S);
        cout << S << endl;
        i++;
    }
}
```
