# N個のデータの入力

https://paiza.jp/works/mondai/stdin/stdin_n

"N行のデータの入力"と"3つのデータの入力"を組み合わせて考えれば正解です。

<details><summary>Python3 解答</summary><br>

`N = int(input())` で `N` を受け取りますが、使いはしません。

```python
N = int(input())
S = input().split()

for x in S:
    print(x)
```

後半の別解("3つのデータの入力"参照)

```python
for i in range(N):
    print(S[i])
```

```python
print(*S, sep = "\n")
```

```python
print("\n".join(S))
```

全体の別解("3つのデータの入力"参照)

```python
N = int(input())
S = input()

print(S.replace(" ", "\n"))
```

<details><summary>C++ 解答</summary><br>

`cin >> S;` を使うことで、入力を空白や改行で区切って読み取ってくれます。<br>
"N行のデータの入力"と違い、`getline` ではなく `cin >> S;` で入力を受け取るので、`N` の後ろに改行が残っていても大丈夫です。そのため、今回の問題では `getline(cin, dummy);` や `cin.ignore();` は不要です。

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;

    string S;
    for (int i = 0; i < N; i++) {
        cin >> S;
        cout << S << endl;
    }
}
```

while文を使ってもいいです。

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;

    string S;
    int i = 0;
    while (i < N) {
        cin >> S;
        cout << S << endl;
        i++;
    }
}
```
