<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    cout << S.back() << endl;
}
```

別解<br>
`S.size()` は `S.length()` でも可。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    cout << S[S.size() - 1] << endl;
}
```

別解2<br>
`end()` は末尾の次を指すので、-1して最後の文字を取得します。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    cout << *(S.end() - 1) << endl;
}
```

別解3

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;

    char last;
    for (char c : S) {
        last = c;
    }
    cout << last << endl;
}
```

別解4

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    cout << S.substr(S.size() - 1, 1) << endl;
}
```
