# N倍の文字列

https://paiza.jp/works/mondai/d_rank_skillcheck_archive/square

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;
    for (int i = 0; i < N; i++) {
        cout << '*';
    }
    cout << endl;
}
```

別解

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;

    string s(N, '*');
    cout << s << endl;
}
```
