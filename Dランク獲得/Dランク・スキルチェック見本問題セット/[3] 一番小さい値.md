# 一番小さい値

https://paiza.jp/works/mondai/d_rank_skillcheck_sample/min_num

<details><summary>C++ 解答</summary><br>

条件に $`1 ≦ n_1, n_2, n_3, n_4, n_5 ≦ 100`$ とあるので、上限よりも大きい101で変数 `answer` を初期化し、その後入力から数字を受け取って小さい方に更新する、というのを5回繰り返せばいいです。

```cpp
#include <iostream>
using namespace std;

int main() {
    int answer = 101, n;
    for (int i = 0; i < 5; i++) {
        cin >> n;
        answer = min(answer, n);
    }
    cout << answer << endl;
}
```

少し変えた別解<br>
1個目の値を `answer` の初期値としてもいいです。ただし、1個目をすでに入力から読み取っている分、入力から受け取る回数が1回少なくなることに注意してください。

```cpp
#include <iostream>
using namespace std;

int main() {
    int answer, n;
    cin >> answer;
    for (int i = 0; i < 4; i++) {
        cin >> n;
        answer = min(answer, n);
    }
    cout << answer << endl;
}
```

別解2<br>
`min` を使わずに、if文を使ってもいいです。

```cpp
#include <iostream>
using namespace std;

int main() {
    int answer, n;
    cin >> answer;
    for (int i = 0; i < 4; i++) {
        cin >> n;
        if (n < answer) {
            answer = n;
        }
    }

    cout << answer << endl;
}
```

別解4-1<br>
`vector` に対して `min_element` を使う。<br>
`vec.at(i)` は `vec[i]` でも可。

```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main() {
    vector<int> vec(5);
    for (int i = 0; i < 5; i++) {
        cin >> vec.at(i);
    }

    cout << *min_element(vec.begin(), vec.end()) << endl;
}
```

#include <iostream>

}
