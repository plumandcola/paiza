# 一番小さい値

https://paiza.jp/works/mondai/d_rank_skillcheck_sample/min_num

<details><summary>C++ 解答</summary><br>

条件に $`1 ≦ n_1, n_2, n_3, n_4, n_5 ≦ 100`$ とあるので、上限よりも大きい101で変数 `answer` を初期化し、その後入力から数字を受け取って小さい方に更新する、というのを5回繰り返せばいいです。

```cpp
#include<iostream>
using namespace std;

int main() {
    int answer = 101, n;
    for (int i = 0; i < 5; i++) {
        cin >> n;
        answer = min(answer, n);
    }
    cout << answer << endl;
    return 0;
}
```

少し変えた別解<br>
1個目の値を `answer` の初期値としてもいいです。ただし、1個目をすでに入力から読み取っている分、入力から受け取る回数が1回少なくなることに注意してください。

```cpp
#include<iostream>
using namespace std;

int main() {
    int answer, n;
    cin >> answer;
    for (int i = 0; i < 4; i++) {
        cin >> n;
        answer = min(answer, n);
    }
    cout << answer << endl;
    return 0;
}
```

別解2<br>
`min` を使わずに、if文を使ってもいいです。

```cpp
#include <iostream>
using namespace std;

int main() {
    int answer, n;
    cin >> answer;

    for (int i = 0; i < 4; i++) {
        cin >> n;
        if (n < answer) {
            answer = n;
        }
    }

    cout << answer << endl;
}
```

別解3<br>
配列を使う

```cpp
#include <iostream>
using namespace std;

int main() {
    int a[5];
    for (int i = 0; i < 5; i++) {
        cin >> a[i];
    }

    int answer = a[0];
    for (int i = 1; i < 5; i++) {
        answer = min(answer, a[i]);
    }

    cout << answer << endl;
}
```

```cpp
#include <iostream>

int main() {

        answer = min(answer, v[i]);
    }

    cout << answer << endl;
}
```

別解5<br>
sortする

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int a[5];
    for (int i = 0; i < 5; i++) {
        cin >> a[i];
    }
    
    sort(a, a + 5);
    cout << a[0] << endl;
}
```
