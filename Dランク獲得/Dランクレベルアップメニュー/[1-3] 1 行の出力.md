# 1 行の出力

https://paiza.jp/works/mondai/d_rank_level_up_problems/d_rank_level_up_problems__stdout_3

<details><summary>Python3 解答</summary><br>

整数は `"` や `'` で囲う必要はありません。

```python
print(813)
```

`"` や `'` で囲っても問題はありません。

```python
print("813")
```

<br></details>

<details><summary>C++ 解答</summary><br>

整数は `"` や `'` で囲う必要はありません。

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << 813 << endl;
}
```

`"` で囲っても問題はありませんが、`'` では囲えません。C++で `'` で囲えるのは1文字の場合のみです。

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "813" << endl;
}
```
