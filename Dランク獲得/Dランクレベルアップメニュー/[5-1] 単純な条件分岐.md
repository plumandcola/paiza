# 単純な条件分岐

https://paiza.jp/works/mondai/d_rank_level_up_problems/d_rank_level_up_problems__conditions_branch_1

<details><summary>ヒント</summary><br>

条件分岐には**if文**を用いましょう。

<br></details>

<details><summary>Python3 解答</summary><br>

`"paiza"` と `"` で囲うこと、`=` ではなく `==` とすること、`:` を忘れないこと、インデントを忘れないことなどに注意しましょう。

```python
S = input()

if S == "paiza":
    print("YES")
else:
    print("NO")
```

以下のように三項演算子を用いると、短くなります。

```python
S = input()
print("YES" if S == "paiza" else "NO")
```

(読みにくいので避けるべきですが、)頑張れば1行で書けます。

```python
print("YES" if input() == "paiza" else "NO")
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    if (S == "paiza") {
        cout << "YES" << endl;
    } else {
        cout << "NO" << endl;
    }
}
```

if文の中身が1行なら `{}` は不要です。しかし読みにくいので避けましょう。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    if (S == "paiza") cout << "YES" << endl;
    else cout << "NO" << endl;
}
```

Pythonと同様、以下のように三項演算子を用いても短くなりますが、やはり読みにくいので避けるべきです。

```cpp
#include <iostream>
using namespace std;

int main() {
    string S;
    cin >> S;
    cout << (S == "paiza" ? "YES" : "NO") << endl;
}
```
