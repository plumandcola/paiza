# 数値の分岐

https://paiza.jp/works/mondai/d_rank_level_up_problems/d_rank_level_up_problems__conditions_branch_2

<details><summary>Python3 解答</summary><br>

```python
N = int(input())

if N <= 100:
    print("YES")
else:
    print("NO")
```

三項演算子ver.

```python
N = int(input())
print("YES" if N <= 100 else "NO")
```

(読みにくいので避けるべきですが、)頑張れば1行で書けます。

```python
print("YES" if int(input()) <= 100 else "NO")
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;
    if (N <= 100) {
        cout << "YES";
    } else {
        cout << "NO";
    }
}
```

三項演算子ver.

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;
    cout << (N <= 100 ? "YES" : "NO") << endl;
}
```
