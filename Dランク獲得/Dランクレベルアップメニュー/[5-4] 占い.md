前回までの `"YES"`, `"NO"` と違い、`"Yes"`, `"No"` なので、大文字・小文字に気をつけましょう。

<details><summary>Python3 解答</summary><br>

```python
n = int(input())

if n == 7:
    print("Yes")
else:
    print("No")
```

三項演算子ver.

```python
n = int(input())
print("Yes" if n == 7 else "No")
```

```python
print("Yes" if int(input()) == 7 else "No")
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    if (n == 7) {
        cout << "Yes" << endl;
    } else {
        cout << "No" << endl;
    }
}
```

三項演算子ver.

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    cout << (n == 7 ? "Yes" : "No") << endl;
}
```
