# 数値演算結果で分岐

https://paiza.jp/works/mondai/d_rank_level_up_problems/d_rank_level_up_problems__conditions_branch_3

<details><summary>Python3 解答</summary><br>

`A, B, C = map(int, input().split())` の部分が初出なので、説明します。<br>
`input()` で入力を読み取る → `input().split()` で、読み取った入力を空白文字で区切ってlist型にする → `map(int, input().split())` で、list型の要素それぞれに関数 `int` をかけてint型にする → `A`, `B`, `C` に代入する

```python
A, B, C = map(int, input().split())

if A * B <= C:
    print("YES")
else:
    print("NO")
```

三項演算子ver.

```python
A, B, C = map(int, input().split())
print("YES" if A * B <= C else "NO")
```

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    int A, B, C;
    cin >> A >> B >> C;
    if (A * B <= C) {
        cout << "YES" << endl;
    } else {
        cout << "NO" << endl;
    }
}
```

三項演算子ver.

```cpp
#include <iostream>
using namespace std;

int main() {
    int A, B, C;
    cin >> A >> B >> C;
    cout << (A * B <= C ? "YES" : "NO") << endl;
}
```
