# 2 つの数値を出力

https://paiza.jp/works/mondai/d_rank_level_up_problems/d_rank_level_up_problems__stdout_boss

<details><summary>Python3 解答</summary><br>

"2 つの文字列を出力"で説明した通り、Pythonの `print` に複数の引数を渡した際は、デフォルトで半角スペースで区切って出力に表示します(デフォルトが `sep=' '` となっている)。

```python
print(1, 1)
```

```python
print("1 1")
```

公式の解答コード例は難しい…。フォーマット関数というものを使っている。文字列内の `{}` ("置換フィールド"と呼ぶ)の部分に、`.format()` の `()` 内のものが入って出力に表示される。

```python
print("{} {}".format(1, 1))
```

"f文字列"も使える。

```python
print(f"{1} {1}")
```
