# ある数字までの出力 2

https://paiza.jp/works/mondai/d_rank_level_up_problems/d_rank_level_up_problems__loop_1

いきなり2？と思ったが、"Cランク獲得" > "ループメニュー1" (https://paiza.jp/works/mondai/loop_problems) 内に"ある数字までの出力 1"と"ある数字までの出力 2"(今回と同じ問題)があるからのようです。

<details><summary>Python3 解答</summary><br>

関数 `range` を使いますが、`range(N)` は **0 から $`N`$-1** までであることに気をつけてください。

```python
N = int(input())

for i in range(N):
    print(i+1)
```

```python
N = int(input())

i = 0
while i < N:
    i += 1
    print(i)
```

`range` を `*` で展開することもできます。<br>
1から $`N`$ までにするために、`range(1, N+1)` としています。

```python
N = int(input())
print(*range(1, N+1), sep = "\n")
```

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;
    for (int i = 1; i <= N; i++) {
        cout << i << endl;
    }
}
```

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;
    int i = 0;
    while (i < N) {
        i++;
        cout << i << endl;
    }
}
```
