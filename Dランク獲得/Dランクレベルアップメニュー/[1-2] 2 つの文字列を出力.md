# 2 つの文字列を出力

https://paiza.jp/works/mondai/d_rank_level_up_problems/d_rank_level_up_problems__stdout_2

<details><summary>Python3 解答</summary><br>

```python
print("paiza learning")
```

以下の書き方では、`"paiza"` と `"learning"` を 別々の引数として `print` に渡しています。<br>
Pythonの `print` に複数の引数を渡した際は、デフォルトで半角スペースで区切って出力に表示します(デフォルトが `sep=' '` となっている)。

```python
print("paiza", "learning")
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "paiza learning" << endl;
}
```
