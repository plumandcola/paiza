# 1 つの文字列を出力

https://paiza.jp/works/mondai/d_rank_level_up_problems/d_rank_level_up_problems__stdout_1

<details><summary>Python3 解答</summary><br>

"期待する出力"に"末尾に改行を入れ、…"とありますが、Pythonの `print()` ではデフォルトで末尾に改行が入る(デフォルトが `end='\n'` となっている)ため、特別に必要なことはありません。

```python
print("paiza")
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "paiza" << endl;
}
```
