# 3 行の出力

https://paiza.jp/works/mondai/d_rank_level_up_problems/d_rank_level_up_problems__stdout_4

<details><summary>Python3 解答</summary><br>

```python
print(8)
print(1)
print(3)
```

`sep = "\n"` とすることで、`8`, `1`, `3` を `\n` (改行)で区切って出力に表示しています。

```python
print(8, 1, 3, sep = "\n")
```

```python
print("8\n1\n3")
```

<br></details>

<details><summary>C++ 解答</summary><br>

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << 8 << endl;
    cout << 1 << endl;
    cout << 3 << endl;
}
```

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << 8 << endl << 1 << endl << 3 << endl;
}
```
